## 반복문
>#### 프로그램 수행 흐름을 바꾸는 역할을 하는 제어문 중 하나로 특정 문장들을 반복해서 수행하도록 함
 - ## 반복문의 종류
 	- #### 1. for문
 ```Java
 for(/*초기식;*/ /*조건식;*/ /*증감식*/) {
 	// 수행될 코드;
 }
 ```
 
	- #### 1. while문
 ```Java
 while(/*조건식*/){
 	//수행될 코드;
 	//[증감식 or 분기문];
 }
 ```
 
 - ## for문
 	- for
 	  ```Java
 	  for(/*초기식;*/ /*조건식;*/ /*증감식*/) {
 	  	// 수행될 코드;
 	  }
 	  ```
